zsh
===

Adds [dotzsh][1] configuration

To use, add the following to **dotsyncrc**

    [files]
    ..
    zsh/dotzsh:.zsh
    zsh/zshrc:.zshrc
    zsh/zshenv:.zshenv
    zsh/zprofile:.zprofile
    zsh/zlogin:.zlogin
    zsh/zlogout:.zlogout
    ..
    [endfiles]

[1]: https://github.com/dotphiles/dotzsh
